## Tamaños

| Componente | Tamaño |
| --- | --- |
| Bus de datos | 16 bits |
| Bus de direcciones | 16 bits |
| Bus de direcciones | 7 bits |
| PC | 7 bits |
| ALU REG. A | 16 bits |
| ALU REG. B | 16 bits |
| RI SRC | 7 bits |
| RI DST | 7 bits |
| RI COD. OP. | 2 bits |

## Registros

La MS tiene un total de 12 registros:

- Registros de control y direccionamiento: PC, RI
- Registros auxiliares de la ALU: A, B
